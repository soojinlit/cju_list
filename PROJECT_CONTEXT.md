# Project Context (CJU신청도서)

## 프로젝트 경로
- `D:\Download\tbr-web`

## 목적
- `2026_tbr.md` 기반으로 만든 개인 도서 목록 웹앱.
- 모바일 접근 가능, 로컬/깃허브 저장 가능.

## 현재 상태 요약
- **표지 관련 기능 전부 제거됨** (표지 표시/검색/ISBN/후보 UI 없음)
- **상세 모달에서만 기록 편집**: 제목/저자/상태/별점(0.5 단위)/읽은 날/메모
- **삭제는 상세 모달에서만 가능**
- **별점**: 0.5 단위 선택, hover 미리보기, 별점 리셋 버튼 있음
- **정렬**: 기본/제목/저자/별점 높은 순/별점 낮은 순
- **필터**: 상태 필터 + 별점 최소 필터(예: 4.5+)
- **페이지네이션**: 이전/다음 + 페이지 크기 30/50/100
- **하단 바**: sticky, 카드/컨테이너 너비와 정렬됨

## UI 문구
- 제목: `CJU신청도서`
- 부제: `도서관에 신청한 것 알차게 다 읽어보자!`

## 주요 파일
- `index.html`: UI 구조
- `styles.css`: 스타일
- `app.js`: 로직
- `data/books.json`: 데이터
- `scripts/convert_tbr_to_json.py`: 마크다운 변환

## 실행 방법
```powershell
cd D:\Download\tbr-web
python -m http.server 8080
```
접속: `http://localhost:8080`

## 주의 사항
- 표지 기능은 제거되었으므로 관련 UI/코드가 없어야 함.
- 상세 모달에서만 기록 편집 가능하도록 유지.
